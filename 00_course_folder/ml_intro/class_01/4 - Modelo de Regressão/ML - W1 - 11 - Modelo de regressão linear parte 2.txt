O algoritmo de aprendizado supervisionado inserirá um conjunto de dados e , em seguida, o que exatamente ele faz e o que ele produz? Vamos descobrir neste vídeo. Lembre-se de que um conjunto de treinamento em aprendizado supervisionado inclui os recursos de entrada, como o tamanho da casa, e também as metas de produção, como o preço da casa. As metas de saída são as respostas certas para o modelo com o qual aprenderemos. Para treinar o modelo, você alimenta o conjunto de treinamento, tanto os recursos de entrada quanto as metas de saída, para seu algoritmo de aprendizado. Então, seu algoritmo de aprendizado supervisionado produzirá alguma função. Escreveremos essa função como f minúscula, onde f significa função. Historicamente, essa função costumava ser chamada de hipótese, mas vou chamá-la de função f nesta classe. O trabalho com f é pegar uma nova entrada x e saída e estimar ou uma previsão, que vou chamar de y-hat, e é escrita como a variável y com esse pequeno símbolo de chapéu no topo. No aprendizado de máquina, a convenção é que y-hat é a estimativa ou a previsão para y. A função f é chamada de modelo. X é chamado de entrada ou recurso de entrada, e a saída do modelo é a previsão, y-hat. A previsão do modelo é o valor estimado de y. Quando o símbolo é apenas a letra y , isso se refere ao alvo, que é o valor real real no conjunto de treinamento. Em contraste, y-hat é uma estimativa. Pode ou não ser o verdadeiro valor real. Bem, se você está ajudando seu cliente a vender a casa, bem, o verdadeiro preço da casa é desconhecido até que ele a venda. Seu modelo f, dado o tamanho, gera o preço que é o estimador, ou seja, a previsão de qual será o preço real. Agora, quando projetamos um algoritmo de aprendizado, uma questão chave é: como vamos representar a função f? Ou, em outras palavras, qual é a fórmula matemática que vamos usar para calcular f? Por enquanto, vamos continuar com f sendo uma linha reta. Sua função pode ser escrita como f_w, b de x igual, vou usar w vezes x mais b. Definirei w e b em breve. Mas, por enquanto, saiba que w e b são números, e os valores escolhidos para w e b determinarão a previsão y-hat com base no recurso de entrada x. Esse f_w b de x significa que f é uma função que toma x como entrada e, dependendo dos valores de w e b, f produzirá algum valor de uma previsão y-hat. Como alternativa para escrever isso, f_w, b de x, às vezes escrevo apenas f de x sem incluir explicitamente w e b em um subscrito. É apenas uma notação mais simples que significa exatamente a mesma coisa que f_w b de x. Essa linha reta é a função linear f_w b de x igual a w vezes x mais b. Ou, mais simplesmente, podemos descartar w e b e simplesmente escrever f de x igual a wx mais b. Aqui está o que essa função está fazendo, ela está fazendo previsões para o valor de y usando uma função simplificada de x. Você pode perguntar: por que estamos escolhendo uma função linear, onde função linear é apenas um termo sofisticado para a linha reta em vez de alguma função não linear, como uma curva ou uma parábola? Bem, às vezes você também quer ajustar funções não lineares mais complexas, como uma curva como essa. Mas como essa função linear é relativamente simples e fácil de trabalhar, vamos usar uma linha como base que acabará por ajudá-lo a chegar a modelos mais complexos que não são lineares. Esse modelo em particular tem um nome, é chamado de regressão linear. Mais especificamente, isso é regressão linear com uma variável, onde a frase uma variável significa que há uma única variável de entrada ou característica x, ou seja, o tamanho da casa. Outro nome para um modelo linear com uma variável de entrada é regressão linear univariada, onde uni significa um em latim e onde variável significa variável. Univariada é apenas uma maneira sofisticada de dizer uma variável.
